## 0. 战略定调(七轮收敛结论)

### 为什么是中东阿联酋(不是新加坡/俄罗斯/伊朗)
| 地区 | 蓝海度 | 结论 |
|---|---|---|
| **阿联酋(UAE)** | ★★★★★ | 三角全满足:需求高(Vision 2030/$400亿)+ 支付摩擦大(信用卡受限)+ 本地合规聚合稀缺。USDT 合法(VARA),香港公司能收。**本月唯一战场** |
| 新加坡 | ★★ | 泛聚合红海(OpenRouter/硅基流动/火山引擎/302.ai),且用户能直付官方,你赚不到钱 |
| 俄罗斯 | ★ | 供给过剩(Sber+Yandex+DeepSeek官方占90%+),SWIFT受阻,非蓝海 |
| 伊朗 | — | OFAC全面制裁(目的地原则,严格责任),法律不可行,**彻底放弃** |
| 沙特(KSA) | ★★★★★(蓝海)但本月不碰 | mada需SAMA牌照(约90天审批),外资本月接不通。第二个月用partner解决 |

### 赚钱逻辑(绕开支付死结)
中东开发者付不了官方的钱(信用卡受限/美元难付)→ 你用他们能付的方式收款:
- **USDT(主通道)**:阿联酋合法,香港合规收,0.5%费率,绕开mada/Stripe/牌照一切死结 ← 本月生命线
- **Stripe香港(补充)**:收国际卡,覆盖约30%能办卡用户,费率3.9%+
- **本地partner持牌网关(第二月)**:覆盖mada的70%用户

### 一个月能赚到的钱的量级(诚实预期)
- 10-30个付费充值用户 × 平均$30-50客单 = **$300-$1500首月现金收入**
- 这不是暴富,是"模式跑通 + 见到第一笔钱"的信号。放量是第二个月起。
- ToB大单不计入本月(周期8-16周),并行培育,出单算超额。

## 2. 一个月赚钱验收标准(可判定成败)

### ✅ 必要(算"赚到钱")
- ≥ 10个付费充值用户 **或** 到账收入 ≥ $500等值

### ✅✅ 充分(算"超出预期")
- 进入复购(用户二次充值)
- 或 ≥ 1个ToB会议/POC(并行培育出成果)

### 明确不算失败
- 当月没签ToB大单(周期8-16周,签约是第二个月目标)

### ❌ 失败信号(任一出现需复盘)
1. Phase 0收款两周接不通 → 地基问题
2. 推广两周后注册→充值转化率 < 2% → 痛点/触达/定价错
3. 中东开发者明确反馈"我们其实能付官方" → 需求假设被证伪
4. USDT充值无人用 → 通道不成立,需评估Stripe-only可行性
5. TG群被打上spam标签 → 口碑问题,需调整策略

### 关键监控指标(每周看)
- 注册数 / 注册→充值转化率(行业基准5-15%)
- 充值金额 / 客单价
- 各渠道ROI(哪个渠道来的用户充值率高)
- TG响应时效

## 5. 风险登记与应对

| 风险 | 概率 | 应对 |
---|---|---|
| USDT开户/KYC受阻 | 中 | Week 1优先解决,备用方案:Cryptomus/Plisio |
| 中东开发者不用USDT | 低中 | A/B测试,退回Stripe-only(覆盖面降30%但能跑) |
| TG群被标spam | 中 | 价值优先,每发1推广先发3纯技术内容 |
| 上游模型来源被质疑 | 中 | 备齐合规举证资产(DPA/授权证明/数据流转图) |
| 支付/数据合规(PDPL) | 中 | landing+充值页明确数据不训练+本地化口径 |
| 口碑被恶意差评 | 低 | 建TG响应机制,快速解决问题 |
