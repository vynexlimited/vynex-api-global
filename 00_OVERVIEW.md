## 一、产品与赛道判断（先认清现实）

你做的不是新东西，而是一个**高度成熟、竞争极其激烈**的赛道：LLM API Gateway / Aggregator。

### 全球主要对手
| 对手 | 优势 | 你的差距 |
|---|---|---|
| **OpenRouter** | 绝对 leader，品牌 + 网络效应 + 极全模型 + 价格透明 | 品牌知名度、信任度 |

**结论**：你不能靠"我也是一个 API 中转"打赢。必须有一个**清晰、可验证、能一句话讲清的差异化卖点**，否则推广出去的用户也留不住。

## 二、推广前必须解决的 3 个硬伤（不解决别推广）

这是看了官网（截至 2026-06）后的判断。这三点不补，推广预算和精力会大量浪费：

### ✅ 已具备（核实源码 vynex-api-homepage 后更正）
- **多语言**：已支持 `en` / `zh` / `ru` 三语，**默认英文**，浏览器自动检测 + 手动切换，英文翻译完整真实（非占位）。
- **定价页**：`/pricing` 已存在并真实展示每个模型的 input/output 价格（动态拉取），导航栏有入口。
- **功能完整**：about / intent / docs / playground / wallet / usage-logs / subscriptions / redemption-codes 等齐全，非简陋中转站。
> 之前误判「只有中文 / 没有定价页」，是因为抓取工具的语言检测把首页渲染成了中文。实际默认语言是英文。

### 🟠 硬伤 1（修正后）：多语言的 SEO / GEO 可见性缺陷（真正要修的）
- 现状：语言切换**只存 localStorage**，URL 不变；首页是 React SPA（客户端渲染），无 SSR。
- 问题：Google / ChatGPT / Perplexity 等**爬虫抓到的是同一 URL**，拿不到带英文内容的可索引 HTML —— 英文内容对搜索和 AI 引擎**基本不可见**。这就是为什么我抓首页时会被识别成中文。
- **必须做**（按性价比排序）：
  1. **URL 带语言标识**：`/en/`、`/zh/`、`/ru/` 路径前缀（首选）或 `?lang=en`，并在服务端/边缘层读取，让每种语言有独立可索引 URL。
  2. **关键页面预渲染（SSG）或 SSR**：至少首页、`/pricing`、`/about` 做服务端渲染，确保爬虫拿到带内容的 HTML（而非空壳 SPA）。
  3. **加 `hreflang` 标签**：在 `<head>` 里声明 `en` / `zh` / `ru` 对应 URL，告诉搜索引擎多语言版本关系。
  4. **部署 `llms.txt`**（已在本素材包 `geo/assets/llms.txt`）：绕过 SPA 渲染，直接给 AI 爬虫英文产品摘要——**这一步可以立刻做，零开发成本**。

### 🟠 硬伤 2（修正后）：首页缺价格钩子 + 价格透明度可强化
- 现状：`/pricing` 页已有价格，但**首页 hero 区没有**价格信息或「比官方便宜 X%」的钩子。
- 问题：开发者逛首页时看不到价格信号，可能不点进 `/pricing` 就离开。
- **建议做**：
  - 首页加一句明确的定位（如「pay per token, see /pricing」或「up to ~X% cheaper than list」——用真实数字）。
  - `/pricing` 页加「vs 官方列表价」对比列，强化性价比心智。
  - 控制台保留实时余额、用量明细、成本预估（已有 wallet/usage-logs，确认对用户可见）。

### 🔴 硬伤 3：信任与合规（欧美开发者的命门）
- 现状：官网/文档没有说明数据如何处理、是否记录、合规资质。
- 问题：欧美开发者把**生产代码和用户数据**经过你的中转层，会直接问：数据存哪？是否留存 prompt？GDPR？会不会泄露我的 key？
- **必须做**：
  - 文档加一页 `Security & Privacy`：明确「不记录 prompt/响应内容（或说明仅记录用量元数据）」「key 加密存储」「数据驻留区域」。
  - 加 `Terms` / `Privacy Policy` / `DPA`（数据处理协议）页面——这些是 B2B 和欧美企业采购的硬门槛。
  - 有条件就上 SOC 2 / ISO 27001 路径（至少声明在推进）。
  - 上游合规：说清楚与 OpenAI/Anthropic/Google 的关系，避免被质疑是"盗版中转"。

## 三、选定一个差异化主卖点（必须选，别贪多）

推广的核心是「用一句话让人记住你」。下面是可选方向，**强烈建议选 1 个为主、其余为辅**：

| 方向 | 卖点话术 | 适合人群 | 可信度要求 |
|---|---|---|---|
| **A. 价格最便宜** | 「比 OpenAI 官方便宜 up to 50%，一个 Key 用全部模型」 | 独立开发者、学生、成本敏感团队 | 必须价格表透明且真实 |
| **B. 无地区限制** | 「在全球任何地方统一调用 GPT/Claude，无需美区账号/信用卡」 | 中国、俄罗斯、中东、东南亚、拉美开发者 | 必须真的稳定可用 |
| **C. 模型最全最快** | 「第一时间接入最新模型（GPT-5.5 / Claude Opus 4.8 等），OpenAI 兼容」 | 尝鲜型开发者、Agent 开发者 | 必须新模型确实可用 |
| **D. 中文开发者友好** | 「中文母语文档、中文客服、人民币支付，但能力对标 OpenRouter」 | 中文开发者出海 / 华人圈 | 较低，但天花板也低 |
| **E. 企业级稳定** | 「99.9% SLA、私有部署、企业账单」 | 企业、B2B | 需要真实基础设施支撑 |

> 我的建议：面向**全球**，主攻 **A（价格）+ C（模型全/新）**，用 **B（无地区限制）** 作为隐性杀手锏（针对 OpenAI 难以触达地区的开发者，这部分人转化率极高且忠诚）。
